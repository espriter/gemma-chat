최근 항공기

최근 N분 동안 수신된 항공기 위치 데이터
최근 30분 내 항공기 20개 보여줘

고유 항공기

기간 내 관측된 고유 항공기 목록 + 메시지 수, 고도 범위, 평균 속도
최근 1시간 내 고유 항공기 목록

주간 트래픽

일별 총 메시지 수, 고유 항공기 수, 평균/최대 고도, 평균 속도
최근 7일 트래픽 요약

가장 먼 항공기

안테나 기준 Haversine 거리로 가장 먼 항공기
최근 24시간 내 가장 먼 항공기 10개

항공기 검색

특정 ICAO hex 코드로 위치 이력 추적
71BE07 최근 24시간 위치 이력

항공기 정보 조회

ICAO hex → 등록번호, 제조사, 기종, 운영사 (hexdb.io)
71BA12 항공기 정보 알려줘

날씨

현재 날씨 + 3일 예보 (wttr.in)
Seoul 날씨 알려줘

퀵 버튼 API 직접 호출

curl로 직접 도구를 호출하는 예시 (LLM 없이 즉시 결과)
# 최근 항공기
curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"recent_aircraft","args":{"minutes":30,"limit":20}}'

고유 항공기 API

curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"unique_aircraft","args":{"hours":1}}'

주간 트래픽 API

curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"traffic_summary","args":{"days":7}}'

가장 먼 항공기 API

curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"farthest_aircraft","args":{"hours":24,"limit":10}}'

항공기 검색 API

curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"search_aircraft","args":{"hex_ident":"71BE07","hours":24}}'

항공기 정보 API

curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"lookup_aircraft","args":{"hex_ident":"71BA12"}}'

날씨 API

curl -s -X POST http://localhost:8095/api/tool \
  -H "Content-Type: application/json" \
  -d '{"name":"get_weather","args":{"location":"Seoul"}}'
